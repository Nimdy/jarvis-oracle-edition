Distillation Flow & Metrics

How JARVIS outgrows its shell: baseline models/LLM act as teachers → fidelity-scored buffers (low-fidelity quarantined) → student-NN training → tier-1 gating (accuracy floor + regression cooldown) → promotion only when a student beats the baseline, at which point it replaces the baseline at inference. "active networks" = operational student specialists; whether each one beats the baseline is its accuracy in the NN Inspector above (e.g. memory 0.95 vs mood 0.38). Promotion-over-baseline is the move off the LLM — per-network, not a single count.
