Brain Responsive

The kernel heartbeat (/api/status .tick) must increment between polls — a frozen tick means the engine is hung even if the HTTP layer answers. Process flags fail closed: anything other than explicit true renders UNKNOWN, never green.

Kernel Cycle Performance

Live observed ticker metrics — these are reports, not targets. avg/p95 are under the <10ms interactive budget when green. budget_overruns and slow_ticks are hard deadline misses, not warnings: any value >0 shows as a red count. max_tick_ms is a single spike — shown raw, never smoothed.

Subsystem Health

Per-subsystem liveness from /api/health/detailed. Each is a fail-closed gate: healthy:true ⇒ ok, false ⇒ fail, missing ⇒ UNKNOWN. Detail strings are the subsystem's own report, shown verbatim.

Host Sensor Health · pi5-senses

A window into the host machine, not the brain. CPU temp is a physical constraint (RPi5 throttles ~80°C); throttled≠0x0 is a hard gate. camera_fps=0 is normal when sleeping. Stale if last read >30s ago.
