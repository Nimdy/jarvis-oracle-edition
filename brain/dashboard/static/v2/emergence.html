Evidence Ladder · Levels 0–7

Eight rungs, ascending. Each carries a support marker (supported / partial / not claimed), an evidence count, representative examples, and — critically — the brain's own limitations and falsification notes. Level 7 (strong anomaly) is deliberately not claimed: nothing has survived known-mechanism elimination. Counts are evidence tallies, not scores; zero at the top is honest restraint, not a gap.

Known-Mechanism Exclusions

Before any rung counts, ordinary explanations must be ruled out. These are the mundane mechanisms the detector treats as disqualifying — an event explained by any of them is not emergence. Level 7 requires an event that survives all of them, reproducibly.
