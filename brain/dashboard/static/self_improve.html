Holographic Cognition / HRR-VSA PRE-MATURE

P4: Holographic Reduced Representations (HRR / VSA) research lane. A derived neural-intuition substrate for compositional internal representations (subject ⊗ relation ⊗ object). HRR is not canonical truth, does not write beliefs directly, does not influence policy until promoted through the normal specialist lifecycle and operator approval under Phase 6.5 governance.

Non-negotiable boundaries (all enforced by test):

  • HRR side-effect counter hrr_side_effects must equal 0 in Stage 0 and Stage 1.
  • No writes to persistent memory, identity state, conversation history, TTS, transcription, belief graph, world-model durable state, or autonomy state.
  • Raw HRR vectors must never enter the LLM articulation layer. Only deterministic human-readable summaries (e.g. cleanup metrics) may.
  • Specialist promotion uses the canonical lifecycle: CANDIDATE_BIRTH → PROBATIONARY_TRAINING → VERIFIED_PROBATIONARY → BROADCAST_ELIGIBLE → PROMOTED (or RETIRED).

Staged plan:

  1. P4.0 — primitive library (brain/library/vsa/*) + symbol dictionary + cleanup memory + synthetic capacity exercise + truth-boundary test. First sprint.
  2. P4.1 — world-state shadow encoder (pure function, no mutation).
  3. P4.2 — mental simulation shadow traces (simulator output unchanged).
  4. P4.3 — HRR-assisted recall advisory (candidate overlap measured; ranker not influenced).
  5. P4.4 — HRR specialist candidate (Tier-1 hrr_encoder) through real lifecycle.
  6. P4.5 — dashboard + API surface (/api/hrr/status).
  7. P4.6 — validation-pack integration (hrr_primitive_library, hrr_truth_boundary, hrr_world_shadow, hrr_recall_advisory, hrr_policy_non_influence).
  8. P4.7 — long-run promotion study (≥ 7 days) with operator-approved promotion only.

Plan: docs/plans/p4_holographic_cognition_vsa.plan.md. Governance rules: AGENTS.md → "HRR / VSA Governance Rules". Science writeup: science → Holographic Cognition.
