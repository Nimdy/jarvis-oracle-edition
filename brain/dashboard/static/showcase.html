▶ NN Maturity Flywheel The path from shadow specialists to governed local authority

The baseline LLM is an articulation surface and teacher signal, not the source of operational truth. The maturity flywheel operates in four stages:

  1. Structured routing — Deterministic routes, registries, and capability gates protect truth boundaries while teacher models produce training data.
  2. Shadow — Hemisphere NNs run in parallel, comparing their output to the teacher's. Accuracy tracked but not acted upon.
  3. Promote — When shadow accuracy exceeds thresholds (decisive win rate >55%, ≥30% decisive, positive margin), the NN takes over live decisions. Teacher becomes fallback only.
  4. Compound — Promoted NNs produce training data for downstream specialists. Each promotion unlocks new learning opportunities.

▶ Identity Fusion Pipeline perception/identity_fusion.py · perception/speaker_id.py · perception/face_id.py

Voice and face signals fuse into canonical IDENTITY_RESOLVED events. Agreement → verified. Disagreement → hold then pick higher confidence. EMA-smoothed scores (alpha=0.35) reduce single-frame volatility.

Recognition state machine: absent → unknown_present → tentative_match → confirmed_match. Cold-start boost: reduced thresholds for 15s after presence arrival.

Multi-speaker awareness: Voice-only promotion suppressed when multiple persons visible and voice confidence < 0.55. Voice trust states: trusted / tentative / degraded / conflicted / unknown.

Identity reconciliation: Voice commands ("merge X into Y", "forget X") perform weighted-average embedding merge, evidence transfer, memory re-tagging, and alias tombstone writing.

▶ Identity Boundary Engine (Layer 3) identity/boundary_engine.py

Retrieval policy matrix controls what memories can be accessed based on who is asking: allow, block, or allow_if_referenced. Prevents cross-identity memory leaks — information shared by User A is not surfaced when User B is the active speaker unless explicitly referenced.

Ring-buffer audit trail tracks scope assignments, boundary blocks, quarantine decisions, and referenced-subject exceptions.
